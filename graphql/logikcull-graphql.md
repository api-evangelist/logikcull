# Logikcull GraphQL API

Logikcull is a cloud-based e-discovery platform. The API covers upload management, processing status, project management, search queries, document review workflows, tag management, production, and analytics.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Logikcull REST API
