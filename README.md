# Logikcull

Cloud eDiscovery platform with a REST API for uploading case documents, managing review projects, applying tags and redactions, and generating production sets. Logikcull enables legal teams to process, review, and produce data in a self-service model with integrations for Slack, Microsoft 365, Google Workspace, Box, and Dropbox.

## About

Logikcull is an instant eDiscovery platform acquired by Reveal Data. It offers cloud-based legal document processing, review, tagging, redaction, and production for law firms, corporate legal teams, and government agencies. The platform is built on AWS infrastructure and supports multipart document uploads via browser-to-S3 technology.
